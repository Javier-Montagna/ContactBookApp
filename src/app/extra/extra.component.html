Description of some of the technologies and package that were used!

Visual studio code: IDE used for development


Angular CLI: Command line interface for crating boilerplate code


bootstrap: Bootstrap v3 standard library for look and feel


express: Small JS server used for serving the data


lodash: JS library used for the data handeling and convertion


ngx-bootstrap: Bootstrap components adapted for angular


redux and angular-redux/store: Original redux implementation adapted to angular for JS store


redux-logger: Middleware in charge of logging all dispatched actions
Solution explained

Angular CLI was used to generate the boiler plate of the application and then the folder structure was created

CONTACTBOOKAPP
  • dist: PROD Version of the code
  • src:
    • app: Holds the entire application
      • contact-book: Hold the component that shows the main contact book
      • contact-detail: Holds the component that shows the detail contact information
      • extra: Has this page on it :P
      • pipes: Holds the custom pipes for phone and address customizatoin
    • assets: Holds all imgs
    • server: Holds the express server used for serving the data
Known Issue
When trying to access the amazon server from localhost to fetch the data a CORS issue was found, to bypass this an express server was created and on it a get method was define to pick up the data on a local file and send it
Running the code
For executing the code please follow these steps:
  1. Clone or download the following repo: https://github.com/Javier-Montagna/ContactBookApp.git
  2. Open the code (For example with visual studio code)
  3. Execute npm install
  4. Execute ng serve/npm start
  5. Open a second terminal
  6. Go to ContactBookApp\src\server
  7. Execute node .\server.js (You should see: the server is available on http://localhost:9001/)
  8. Visit http://localhost:4200 and use the app
If you want to ran the PROD build follow these steps:
  1. Clone or download the following repo:
  2. Open the code (For example with visual studio code)
  3. Execute npm install
  4. Execute ng build --prod --aot
  5. Execute npm install http-server -g
  6. Go to ContactBookApp\src\server
  7. Execute node .\server.js (You should see: the server is available on http://localhost:9001/)
  8. Execute http-server ./dist (on the ContactBookApp root folder)
  9. Visit 127.0.0.1:8080 and use the app
